    1.2.1 Short intro to git

    Git is a version control system that tracks changes in files over time. It allows multiple people to collaborate on a project, maintaining a history of changes. Git helps manage different versions of code, facilitates collaboration, and provides tools for branching, merging, and tracking modifications. It ensures a reliable and organized development process, enabling developers to work independently and merge their changes seamlessly.

    1.2.3 Cloning the project

    You have gotten all the Required Programs and set them up now its time to Clone the project.

    Open your Git client of choice and copy the link to the master branch of your current project. In this example here:

    https://git.autobahn.thesecretvault.net/Autobahn-Raser-Team/a2racermvp.git

    Next go to you on your client Clone Reposetroy and insert the link.

    example screen shot

    After it is finished pull your branch.

    1.3 Building and Starting the Project

    Right click on the .uproject file and select Generate Visual Studio project files, this will now generate the required build files for the project. After that has finished Open the .uproject file and it will ask you Would you like to rebuild them now hit yes, if everything goes well Unreal will now build the project automatically for you. This can take a while where nothing happens, just lean back and relax.
